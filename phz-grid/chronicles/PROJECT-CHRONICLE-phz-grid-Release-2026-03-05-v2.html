Key Architecture Principles

1
Accessibility First
Every feature works with screen readers, keyboard navigation, Forced Colors Mode, and motor impairment accommodations. WCAG 2.1 AA compliance. Semantic Shadow DOM for accessible virtualization.
2
Modular by Default
Tree-shakeable modules. Core bundle <50 KB gzipped (achieved: 7.8 KB). Enterprise features lazy-loaded (DuckDB: 3.5 MB on demand).
3
Schema-as-Contract
Grid state API generates JSON Schema that drives AI integration, documentation, and type generation. Provider-agnostic (OpenAI, Anthropic, Google, local models).
4
Three-Model Extensibility
Hooks (before/after middleware), Slots (named Web Component slots), Headless Core (functional composition). Three extension patterns for different needs.

Architecture Decision Records (8)

▼
ID Decision Status Impact
ADR-001 Headless Core + Lit Rendering Separation
Pure business logic in core, framework-agnostic rendering in grid. Enables framework adapters.		 Accepted Foundational
ADR-002 DOM Virtualization Default, Canvas Enterprise
DOM for 10K-500K rows (default), Canvas for 500K-100M (enterprise opt-in), Hybrid mode.		 Accepted Performance
ADR-003 Semantic Shadow DOM for Accessible Virtualization
Parallel a11y tree maintains ARIA structure. First grid with accessible virtualization. WCAG 2.1 AA.		 Accepted Competitive Moat
ADR-004 DuckDB-WASM + Apache Arrow
10-100x faster than JS arrays. SQL interface. Lazy-loaded (3.5 MB WASM binary).		 Accepted Performance
ADR-005 Yjs CRDTs for Collaboration
Conflict-free real-time editing. Peer-to-peer capable. Offline-first architecture.		 Accepted Enterprise
ADR-006 Three-Layer CSS Token System
Primitive → Semantic → Component tokens. 357 total tokens for full theming control.		 Accepted Design System
ADR-007 Schema-as-Contract AI Integration
Provider-agnostic (OpenAI, Anthropic, Google, local). Zod-based schema generation & validation.		 Accepted Innovation
ADR-008 ESM-Only, No CommonJS
Modern module system. Node 20+ required. Better tree-shaking, smaller bundles.		 Accepted Ecosystem
